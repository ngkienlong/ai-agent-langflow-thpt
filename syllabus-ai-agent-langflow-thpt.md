# Syllabus: Xây dựng hệ thống AI Agent tư vấn tuyển sinh với Langflow

**Đối tượng:** Học sinh THPT.
**Số buổi:** 8 buổi × 90 phút.
**Mục tiêu cuối khoá:** Học sinh tự xây dựng một hệ thống AI Agent tư vấn tuyển sinh đại học trên Langflow Desktop, gồm 3 agent phối hợp:

1. **Agent Chiến lược** — Hỏi 8 câu mở đầu cố định, từ đó tự động sinh bảng hỏi cá nhân hoá (mỗi học sinh nhận bảng hỏi khác nhau) và đẩy lên Notion qua **Notion bundle** (component có sẵn) để học sinh trả lời.
2. **Agent Thông tin tuyển sinh** — Trả lời câu hỏi tuyển sinh theo chiến lược ưu tiên RAG trước (AstraDB), nếu không đủ thông tin thì báo user và hỏi có muốn search web uy tín không. Ngoài ra thu thập thông tin từ web/file → lưu AstraDB, và trích xuất thành bảng có cấu trúc đẩy lên Notion qua **Notion bundle**.
3. **Agent Tư vấn** — Nếu có file trả lời bảng hỏi thì đọc và tư vấn ngay. Nếu không có thì hỏi nhanh 8 câu giống Agent Chiến lược. Kết hợp thông tin cá nhân + dữ liệu tuyển sinh để tư vấn chọn trường, ngành nghề, kế hoạch học tập.

**Công cụ:**

- Langflow Desktop (cài trên máy, miễn phí) — [https://docs.langflow.org](https://docs.langflow.org).
- Model: Gemma 4 — free tier, API key từ Google AI Studio (truy cập qua Gemini API) — [https://aistudio.google.com](https://aistudio.google.com).
- Vector Database: AstraDB Serverless (free tier) với **vectorize integration** bật sẵn (Astra tự lo embedding) — [https://astra.datastax.com](https://astra.datastax.com).
- Web Search: component **Web Search** có sẵn trong Langflow (DuckDuckGo / Google News / RSS).
- Notion: workspace miễn phí + **Notion Internal Integration** (integration token) + Notion bundle trong Langflow (các component có sẵn: Add Content to Page, Create Page, List Database Properties, List Pages, Page Content Viewer, Search, Update Page Property, List Users).

**Cơ sở dữ liệu trên Notion (đẩy lên qua Notion bundle components):**

1. **Bảng hỏi học sinh** — Lưu bảng hỏi cá nhân hoá do Agent Chiến lược sinh ra cho từng học sinh.
2. **Bảng tuyển sinh** — Một database duy nhất gộp 5 cột: Trường, Ngành, Tổ hợp môn, Điểm chuẩn tham khảo (năm trước), Học bổng.
